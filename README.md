# tariff-bot-demo

## GitHub Pages setup

1. Open **Settings** → **Pages**.
2. Set **Source** to **Deploy from a branch**.
3. Choose your default branch and set the folder to **`/docs`**.
4. Save.

Expected published URL:

- https://alex-wu-au.github.io/tariff-bot-demo/

## Dialogflow ES web demo

This site embeds the Dialogflow ES Web Demo URL in an iframe:
